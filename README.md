A professional react website with easy to test components. 

This react project uses
* Redux-saga
* Selectors for memoization
* React.memoize,
* PWA,
* firebase,
* stripe api
* styled components

For Graphql visit
 https://github.com/easyrun32/GraphQL-Ecommerce
